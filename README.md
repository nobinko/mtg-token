# MTG Token Finder

フォーマットを選び、直近のデッキリスト掲載ページを巡回して、トークンを出す採用カードを抜き出す配信準備用ツールです。

## 起動

```powershell
npm start
```

ブラウザで以下を開きます。

```text
http://localhost:5177
```

## できること

- Standard / Pioneer / Modern / Legacy を選択
- MTGGoldfish、MTGDecks、magic.gg の初期URLを自動入力
- 巡回元URLを手動で追加/差し替え
- Scryfallでフォーマット合法のトークン生成カード候補を取得
- デッキリスト掲載ページ内に登場するカードだけを表示
- 英語名、日本語名、画像、セット名、Oracle、採用元URLを表示

## 注意

- 初回検索はScryfallとデッキリストサイトへアクセスするため時間がかかります。
- サイト側のHTML構造やアクセス制限によって、巡回できないページがあります。
- トークン種別はOracle文からのヒント表示です。配信前の最終チェックではカード画像やOracle欄も確認してください。
- ハレルヤなど、サイト固有の構造に強く合わせたい場合は `server.mjs` に専用アダプタを追加する想定です。
